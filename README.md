# 🏝️ Treasure Island

🏝️ A text-based adventure game built in Python where players make choices to find the hidden treasure while avoiding dangerous obstacles.

## Features

- Interactive choice-based gameplay.
- Multiple story paths and endings.
- Uses conditional statements to control the game flow.
- Includes ASCII art for a better gaming experience.
- Beginner-friendly Python project.

## How to Run

```bash
python treasure_island.py
```
